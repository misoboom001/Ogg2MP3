# Dependencies
- python3.8  
  [https://peps.python.org/pep-0569/](https://peps.python.org/pep-0569/)
- pydub  
`pip install pydub`
- ffmpeg  
`pip install ffmpeg-python`

# How to use
`[Download_directory]/ogg2mp3.py [The directory path that contains ogg files] `  
Example   
`C:\Download\ogg2mp3.py C:audio\oggfiles`
